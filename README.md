# Ham-Cheeto

## 게임소개

조작법 및 게임 설명



\---



햄치토(Hamcheeto)



토스트집 사장(특:햄스터)이 되어 장사를 하는 게임

조막만한 손을 움직여 열심히 토스트를 만들어보세요!





\---



조작법



마우스 포인트\&클릭으로 게임 진행

각종 재료를 드래그해 그릴에 올려놓으면 조리가 시작된다.

일정 시간이 지나면 익으며, 일정 시간이 더 지나면 타버린다.

조리 시간은 재료의 종류에 따라 다르다.

포장지 위에 원하는 상태의 재료를 올려 토스트를 만든다.

토스트의 가장 아래와 윗장은 빵이 되어야 한다.

소스류(케찹,머스타드)는 포장지 위에 올린 빵에만 짤 수 있다.

완성된 토스트는 Space키를 누르면 포장된다.

ESC를 누르면 다시 시작하거나 게임 방법을 다시 볼 수 있다.



\---



게임 규칙

들어온 주문대로 토스트를 만들어 손님에게 팔고,

목표 금액을 모으면 게임이 클리어된다.

주문 제출에 3번 실패하면 게임 오버!
